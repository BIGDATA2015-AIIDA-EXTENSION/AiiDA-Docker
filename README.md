## Dockerfile for setting up AiiDA + postgres

Dockerfile for setup  of AiiDa + postgres. Automated build.

## Installation

docker run -it skoneka/docker-aiida /sbin/my_init -- bash -l

now run: verdi shell

## Building your own image

docker build -t aiida .
